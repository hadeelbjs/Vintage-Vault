# 🛍️ Vintage Vault

**Vintage Vault** is a frontend web application that provides a platform for users to **buy and sell vintage items** — bringing classic treasures to modern buyers.

---

## 📌 About the Project

Vintage Vault is a course project built for the **Web Technologies** course. It showcases core frontend development skills through a fully functional vintage marketplace interface where users can browse items for sale or list their own vintage finds.

---

## 🚀 Features

- 🛒 **Shop** for a wide variety of vintage items
- 📦 **Sell** your own vintage pieces
- 🎨 Clean and responsive UI design
- ⚡ Lightweight — no frameworks or backends required

---

## 🖥️ Getting Started

Since this is a pure frontend project, no installation or server setup is required.

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/vintage-vault.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd vintage-vault
   ```

3. **Open in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to launch it in your default browser.
