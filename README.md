# 🛍️ Vintage Vault

**Vintage Vault** is a frontend web application that provides a platform for users to **buy and sell vintage items** — bringing classic treasures to modern buyers.

🌐 **Live Demo:** [hadeelbjs.github.io/Vintage-Vault](https://hadeelbjs.github.io/Vintage-Vault/index.html)

---

## 📌 About the Project

> *"Preserving the beauty and craftsmanship of the past, while offering a curated collection that speaks to the modern world."*

Vintage Vault is a course project built for the **Web Technologies** course. It showcases core frontend development skills through a fully functional vintage marketplace where users can browse curated categories, explore special offers, read customer reviews, and list their own vintage finds — all with no backend required.

---

## 🚀 Features

- 🛒 **Shop by Category** — Browse Clothing, Artworks, and Furniture
- 📦 **Seller Dashboard** — List and manage your own vintage pieces
- 🏷️ **Special Offers** — Promotions like 50% off, free shipping, buy-one-get-one deals, and more
- ⭐ **Customer Reviews** — See what other buyers are saying
- 📱 **Responsive Design** — Clean UI that works across devices
- ⚡ **Lightweight** — Pure frontend, no frameworks or backend required

## 🖥️ Getting Started

Since this is a pure frontend project, no installation or server setup is required.

1. **Clone the repository**
   ```bash
   git clone https://github.com/hadeelbjs/Vintage-Vault.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Vintage-Vault
   ```

3. **Open in your browser**

   Simply double-click `index.html` or open it with a browser of your choice.
